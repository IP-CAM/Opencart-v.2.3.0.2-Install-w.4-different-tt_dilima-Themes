  • Install quickstart : By using this package, you will set the theme exactly as our Demo with sample data.

  • Install only template : If you exist a opencart website you can install only template.

Install template

When finish install default opencart, we install template:

  1. Extract the tt_dilima_template_1.0.zip from the archive you downloaded from Themeforest.
  2. Locate the dilima folder . We recommend 2.0.0.0, as it has significant improvements.
  3. Drag the 3 folders (admin, catalog, image) & 4 folders which included xml files over to your OpenCart installation and merge them with the existing ones.
  4. Install Vqmod

    Open your browser -> Type url of your [opencart]/vqmod/install -> If show VQMOD ALREADY INSTALLED! -> That's OK -> Continue.

  5. Login your opencart admin
  6. Go to Extensions -> Modules -> Install Template -> Choose Install button
  7. Go to Tool -> Backup/Restore

    1 - Click "Browse" to choose database in folder database

    2 - Click "Restore" button to import database

  8. Go to Extensions -> Extension Installer -> Upload extension xml files -> Go to Extensions -> Modifications -> Choose all "Modifications Name" -> Click Refresh.

  9. Login your opencart admin

    Go to System -> Settings

    At tab Store Name , choose your default store, click Edit button.

    Choose tab Store

    At Template : Choose template you want to use

    At Default Layout : Choose layout you want to use

    Note :

    If you want your store to like demo, choose :

    - Like demo Dilima 1 : At Template : Choose tt_dilima1, At Default Layout : Choose Home1 Dilima

    - Like demo Dilima 2 : At Template : Choose tt_dilima2, At Default Layout : Choose Home2 Dilima

    - Like demo Dilima 3 : At Template : Choose tt_dilima3, At Default Layout : Choose Home3 Dilima

    - Like demo Dilima 4 : At Template : Choose tt_dilima4, At Default Layout : Choose Home4 Dilima


In Dilima theme, Dilima1 is default theme so you don't need set layout for it. Example: Set your store like Dilima 2 demo:

  1. Set layouts you want to use:

    Login admin-> System -> Design -> Layouts -> Layout name of the current default-theme (Home1 Dilima, Category, Product...) -> Edit -> Remove Route with Store :"Default"-> Save -> Layout name of template (Home2 Dilima, Category1234, Product1234,..) you want to use -> Edit -> Add Route with Store :"Default" and Route like route of that layout -> Save

  2. Show Cms blocks :

    Go to Extensions -> Modules -> Cms block -> Edit -> List Cms -> At List Items, choose cms blocks used in this template -> Choose "Configuration" -> At entry_store : Select Default Store

    Cms blocks used in Dilima2 : Banner Center Home2, Banner Static Home2, Banner Static Top, Banner Left

  3. Show Slide show :

    Go to Extensions -> Modules -> OC Slide show -> Edit-> Slides Manager -> At Banner Name, choose Banner Slider Home1 -> Edit -> At Module -> At slides, at entry_store, selects Default Storeto use our slides.

  4. Refresh home page. Finish installer.

Creating Product Categories

Adding a new Category

  1. Login your opencart admin

  2. Go to Catalog -> Categories -> Categories

  3. Select the Insert button in the upper right corner.

  4. On the General tab, fill in the fields for your new category (you can make changes at any time). Fields that are required are the "Category Name", "Mega Tag Title"

  5. Click on the Data Tab and fill in the fields you want (these are optional).

  6. Click on the Design Tab and select from the dropdown menu if you want (these are optional).

  7. Click Save button when finish.

You can add as many categories as you would like.

Creating Products

Adding a new Product

  1. Login your opencart admin

  2. Go to Catalog -> Products -> Add New

  3. Now you will see seven tabs (General, Data, Links, Attribute, Option, Discount, Special, Image, Reward Points, and Design). Be sure to fill in the Product Name , Meta Tags Title - required fields - on the first tab.

  4. Click Save button when finish.

You can add as many products as you would like.

To set product as Featured

  1. Login your opencart admin

  2. Go to Extensions -> Modules

  3. At module OC Featured Slider > Featured Slider Home1 (or Featured > Feature Products), choose Edit

  4. Start typing product name to Products field. Then please select the Product name to add it to the list of Featured products

  5. Click Save button when finish.

Set Product as Special

  1. Login your opencart admin

  2. Go to Catalog -> Products

  3. Search for the product you would like to set on specials. Click Edit link to edit it.

  4. Choose Special tab and click Add Special button

  5. Special prices change the value of the product price to the new special price. Specials for a customer group will display the special price only if the customer is logged in as that group.

    Customer Group - Optionally select a customer group you want this special to apply to.

    Priority - If you will offer multiple specials and/or discounts, enter 1 for this to be applied first, 2 for this to be applied second.

    Price - The price you want the customer to see.

    Date Start - When you want this special price to appear.

    Date End - When you want this special price to disappear.

  6. Click Save button when finish.

Set product as Discount product

  1. Login your opencart admin

  2. Go to Catalog -> Products

  3. Locate the product you want to create the discount for . Click Edit link to edit it.

  4. Choose Discount tab and click Add Discount button

  5. We got :

    Customer Group - Optionally select a customer group you want this special to apply to.

    Priority - If you will offer multiple specials and/or discounts, enter 1 for this to be applied first, 2 for this to be applied second.

    Price - The price you want the customer to see.

    Date Start - When you want this discount price to appear.

    Date End - When you want this discount price to disappear.

  6. Click Save button when finish.

Set product as Related product

  1. Login your opencart admin

  2. Go to Catalog -> Products

  3. Locate the product you want to add related products to . Click Edit link to edit it.

  4. Choose Links tab and scroll down until you see the Related Products field

  5. Start typing a product name in that field and OpenCart will drop down a list of products that start with whatever letters you have typed in.

    Important! If the product you are typing is not in your OpenCart products, the drop box will not appear. Make sure you are typing a product that you have in your OpenCart.

  6. Click Save button when finish.
